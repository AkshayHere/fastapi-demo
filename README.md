### Fast API Demo

[Reference](https://realpython.com/fastapi-python-web-apis/)

When we have changes to installation, we commit those into `requirements.txt` file.
```bash
pip freeze > requirements.txt
```

When u kick up the application in another laptop, u start with the help of below commands
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
